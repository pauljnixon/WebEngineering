Web Engineering

Lab: National Historic Places

Starting In-Class on Tuesday, October 1st

Overview

In this lab, you will deploy a Rails app to a Platform as a Service (PaaS), Heroku, so your work can be used by the world. In a small team and in a series of challenges, you team will create a web application that locates the nearest historic places within three (3) miles of your location. The list of historic places is provided by the National Register of Historic Places.

Getting Started

  1. Download list of historic places in the northeast region provided by the National Register of Historic Places: ne_landmarks.zip (CSV, comma-separated, file compressed to ZIP). The list has 18,246 records and the CSV file has four (4) columns: longitude, latitude, name of historic place, and info on historic place.

Challenges

  1. Load all the historic landmarks into a database.
  2. Push your application to Heroku.
  3. Display all the historic landmarks on map view.
  4. Display all the historic landmarks on map view quickly for any user.
  5. Display only the historic landmarks on map view within 3 miles of your location.
  6. Enable basic authentication (e.g., some sort of username and password) to use this web application.

Lab: Ruby and CouchDB

Due: Tuesday, September 24th at 11:59 PM

Instructions

CouchDB is very similar to MongoDB, including terminologies used. Using the couchdb gem, write a Ruby program named mycouch.rb that connects to a CouchDB server at 67.23.79.113 port 5984, creates a database that is the same name as your GitHub handle, and inserts a document (or documents) into your database. You must also handle exceptions, such as the inability to connect, gracefully.
